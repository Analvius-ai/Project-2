# 📊 Project Overview
This project is a Financial Performance and Expense Analysis Dashboard developed using Power BI.
The dashboard provides a comprehensive overview of income, expenses, savings, and budget performance to help monitor financial health and control costs.

It enables businesses or individuals to track financial trends, expense categories, and savings performance across different months and years.

# 🎯 Project Objectives
# The main objectives of this dashboard are:
* Analyze total income, expenses, and net savings
* Track monthly financial performance
* Compare income vs expenses
* Identify highest expense categories
* Monitor budget vs actual expenses
* Understand expense distribution across categories
